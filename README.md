# MySQL Basics

Beginner-friendly MySQL notes and exercises for students starting web development.

## Course Material

- [MySQL Basics for Web Students](./MySQL_Basics_for_Web_Students.md)

## What Students Learn

- Relational database fundamentals
- Table design and constraints
- CRUD queries
- Filtering, sorting, grouping
- Joins and relationships
- Transactions and basic security

## How to Use This Repo

1. Read the lesson file from top to bottom.
2. Run each SQL example in your local MySQL environment.
3. Complete the mini project and practice questions at the end.
